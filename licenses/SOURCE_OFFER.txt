Build instructions
------------------

To rebuild the EXE package from source on Windows:

    python -m venv .venv
    .\.venv\Scripts\python.exe -m pip install --upgrade pip
    .\.venv\Scripts\python.exe -m pip install -e ".[dev]"
    .\scripts\build_exe.ps1

This produces:

    dist\HEICConverter
    dist\HEICConverter.zip

Redistribute dist\HEICConverter.zip. Do not redistribute only
HEICConverter.exe, because the license notices and source availability notice
are part of the binary distribution package.
